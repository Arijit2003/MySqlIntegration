
# MySqlIntegration

This project includes mysql integration with android using Volley.



## Features
- LogIn/SignUp page
- Recycler View (to show a list)
- Search button
- Info page
- Add Button(FAB) to add a new item in the list 
- Delete button(It delets an item from the list)
-- Edit a list item.
